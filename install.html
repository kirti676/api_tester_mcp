🧪 API Tester MCP

Comprehensive API testing capabilities for MCP clients

🚀 Quick Install

Install API Tester MCP directly into your MCP client with one click:

Install in VS Code Install in VS Code Insiders

📋 Manual Installation

For other MCP clients, use this configuration:

{ "mcpServers": { "api-tester": { "command": "npx", "args": ["@api-tester/mcp@latest"] } } }

🔧 NPX Usage

You can also run directly with npx:

npx @api-tester/mcp@latest

✨ Features

  • 📥 Swagger/OpenAPI and Postman collection support
  • 🔄 Automatic test generation
  • ⚡ API and load testing
  • 🔐 Authentication support
  • 📊 HTML reports
  • 📈 Real-time progress tracking
